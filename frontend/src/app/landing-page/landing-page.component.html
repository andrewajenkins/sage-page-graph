How it works!
Pop open the explainers for each section of the application and learn what it's used for.
Purpose: The left sidebar allows users to manage their active and past conversations. Users can view a list of available conversations, delete old ones, or start new ones. It provides quick access to conversation history and control over what conversations are displayed.

Key Features:
  • Add New Conversation: Start a new conversation.
  • Delete Current Conversation: Remove an active conversation from the list.
  • Conversation List: View a list of all available conversations, with the current one highlighted.
Purpose: The center panel is a visual representation of the conversation’s branching structure, allowing users to explore different paths. It shows all branches and nodes of the conversation, which can be clicked to navigate through various parts of the discussion. This structure lets users easily jump to any topic or point in the conversation.

Key Features:
  • Interactive Graph: View and interact with the conversation flow.
  • Branch Navigation: Click on any branch or node to navigate to that part of the conversation.
  • Dynamic Exploration: Easily switch between different sections of the conversation.
Purpose:This panel displays the active, selected branch of the conversation, where users interact with the chatbot (e.g., ChatGPT). Users can add to the conversation, ask follow-up questions, and receive responses. The content of this panel updates dynamically when the user clicks on different parts of the graph in the middle panel.

Key Features:
  • Real-Time Chat: Engage with the chatbot by sending queries and receiving responses.
  • Dynamic Content: The conversation displayed updates based on the selected branch from the center panel.
  • Add to Conversation: Input new queries or follow-ups to continue the conversation.
